2 OTP(2단계 인증) 등록 방법

Google Authenticator 앱을 설치한 후 아래 순서대로 진행해 주세요.

📲 앱 설치 (최초 1회)
· iOS : App Store에서 Google Authenticator 앱 설치
· Android : Google Play에서 Google Authenticator 앱 설치
  1. Google 계정에 로그인
    accounts.google.com 접속 후 2차 PoC 계정(@ekdp.com)으로 로그인합니다.
  2. 보안 설정 페이지 이동
    우측 상단 프로필 아이콘 클릭 → Google 계정 관리 → 상단 탭 [보안] 클릭
  3. 2단계 인증 시작
    'Google에 로그인하는 방법' 섹션 → [2단계 인증] 클릭 → [시작하기]
  4. 인증 앱 선택
    인증 방법 목록에서 [인증 앱] 선택 → [계속]
  5. QR 코드 스캔
    화면에 표시된 QR 코드를 Google Authenticator 앱으로 스캔합니다.
    앱 실행 → 우측 하단 [+] 버튼 → [QR 코드 스캔]
  6. 인증 코드 입력 및 완료
    앱에 표시된 6자리 숫자를 화면에 입력 → [확인] 클릭으로 등록 완료
    ⏱ 코드는 30초마다 갱신됩니다. 만료 전 빠르게 입력해 주세요.
✅ 등록 완료 후 다음 로그인부터 비밀번호 + OTP 6자리를 함께 입력하게 됩니다.
